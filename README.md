# Smart Resume Analyzer

## Live Demo
[Open App](https://smart-resume-analyser-eta.vercel.app/)

A full-stack web application that analyzes resumes against multiple job descriptions and provides match scores, missing keywords, and improvement suggestions.

## Features

- Upload Resume (PDF)
- Compare with multiple Job Descriptions
- Company-wise Match Score
- Missing Keywords Suggestions
- Matched Keywords Detection
- Ranked Results
- Dashboard with statistics
- Analysis History
- Delete History
- Authentication (Login / Register)
- Clean modern UI

## How It Works

1. User logs in or registers  
2. Uploads resume (PDF)  
3. Adds one or more job descriptions  
4. System extracts resume text  
5. Compares with each job description  
6. Calculates match score  
7. Shows best match and suggestions  
8. Saves analysis in history
